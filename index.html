Smart USB-TTL Converter

This project outlines the design of an advanced USB-TTL serial converter to simplify development and testing. It integrates intelligent features like automatic baud rate and pin detection, along with wireless connectivity, creating a versatile tool for any electronics workspace.

Core Functionality

Click on each feature to explore the underlying logic and design.

Auto Baud Rate Detection

▼

The device eliminates manual configuration by intelligently identifying the correct transmission speed. The firmware cycles through a list of standard baud rates (from 300 to 115200), listening for data without framing errors. Upon receiving valid data, the rate is locked for stable communication.

Auto RX/TX Pin Swap

▼

To prevent common connection issues, the converter automatically handles incorrect RX/TX wiring. If no data is received on the default pin configuration after a short timeout, the firmware programmatically swaps the RX and TX pin functions and retries, ensuring a connection is established seamlessly.

Wireless Terminal

▼

The device integrates an ESP32 to function as a wireless bridge. In Wi-Fi mode, it hosts a web server with a WebSocket terminal. In Bluetooth mode, it uses the Serial Port Profile (SPP), allowing connection from any standard Bluetooth terminal app on a mobile device or PC.

Key Components

  • Microcontroller: ESP32-WROOM-32
  • USB Bridge: CP2102 or CH340G
  • Regulator: AMS1117-3.3
  • Level Shifter: Optional (BSS138 based)
  • Connectors: USB-C / Micro USB
